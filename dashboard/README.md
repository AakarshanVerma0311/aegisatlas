# AegisAtlas Dashboard

Authority dashboard built with React 18 + Leaflet + Socket.IO.

## Run

```bash
npm install
npm run dev
```

Set `VITE_BACKEND_URL` to your backend base URL if not using localhost.

## Features

- Authority login
- Real-time tourist updates
- Click tourist for direct map focus + breadcrumb trail (last 5 pings)
- Alert sidebar and E-FIR auto-draft trigger
